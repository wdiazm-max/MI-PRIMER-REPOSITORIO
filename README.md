# Mi primer repositorio 
## Análisis de Ventas y Clientes

Este repositorio contiene dos conjuntos de datos principales: uno relacionado con las ventas realizadas y otro con la información de los clientes. Ambos archivos están organizados en formato CSV y pueden utilizarse para análisis exploratorio, visualizaciones, dashboards o modelos de machine learning.

## Archivos Incluidos

### **ventas.csv**
Contiene información detallada sobre transacciones realizadas. Incluye columnas como:
- id_venta
- fecha
- producto
- cantidad
- precio_unitario
- total
- id_cliente

### **clientes.csv**
Contiene datos básicos y demográficos de los clientes, tales como:
- id_cliente
- nombre
- edad
- ciudad
- segmento
